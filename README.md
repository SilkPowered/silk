# Silk 
The next generation of silk.  
Silk is a fabric mod provides compatiblity with spigot plugins.

## Status
- [ ] Spigot without plugins working.
- [ ] Run mods which not using mixin.
- [ ] Print log into IDEA console.
- [ ] Run fabric api.
- [ ] Run mods using Mixin.
